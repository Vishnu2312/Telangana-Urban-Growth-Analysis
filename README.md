# Telangana Urban Growth Analysis (Power BI)

## Project Overview
This project focuses on transforming and visualizing historical census data to analyze urban population distribution and growth trends across Telangana. 

## Key Features & Insights
* **Total Urban Population KPI:** Displays the high-level metrics of the state's total urban footprint.
* **Top 10 Cities Dynamics:** An interactive bar chart tracking the most populated urban hubs, highlighted by Hyderabad's dominant scale.
* **Urban Center Breakdown:** A donut chart classifying the population distribution by administrative status (Municipal Corporations vs. Census Towns).
* **Interactive Filtering:** Includes a fully functional slicer allowing users to filter the entire dashboard by specific local regions dynamically.

## Tools & Skills Used
* **Data Cleaning:** Power Query Editor (handling data types, setting up whole numbers).
* **Data Visualization:** Power BI Desktop (Cards, Clustered Bar Charts, Donut Charts, and Slicers).
* **Analysis:** Data filtering using Top N criteria.

## How to View
* **Quick View:** Open `Telangana_Urban_Growth_Dashboard.pdf` to see the complete dashboard layout instantly.
* **Interactive View:** Download `Telangana_Urban_Growth_Dashboard.pbix` and open it in Power BI Desktop to interact with the filters.
